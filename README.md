### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MSBivens/learn_web3.git
   ```
2. Install pipenv packages
   ```
   pipenv install PyMySQL flask
   ```
3. Activate popenv environment
   ```
   pipenv shell
   ```
4. Install bcrypt
   ```
   pipenv install flask-bcrypt
   ```
5. Run local server
   ```
   python server.py
   ```
